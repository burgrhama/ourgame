🚀 Installation & Run (Desktop)

The game is built with Python and Pygame. Follow these steps to play:

  1. Clone the repository:
    git clone https://github.com/yourusername/pixel-survival-game.git
  2. Install dependencies:
    pip install -r requirements.txt
  3. Run the game:
    python main.py

📁 Project Structure

The code is organized into 9 modular Python files:

  • main.py - Game loop
  • constants.py - Configuration
  • entities.py - Game classes
  • inventory.py - Crafting & items
  • terrain.py - World generation
  • assets.py - Asset loading
  • rendering.py - UI & drawing
  • ui.py - Menus
  • physics.py - Collision detection

📖 How to Play

Getting Started

You start with your bare hands in an infinite procedurally-generated world. Your goal is to survive, gather resources, and build structures!

Mining

Left-click on blocks to break them:

  • Bare Hand: Slow on wood, very slow on stone
  • Axe: 1.25x faster on wood (uses durability)
  • Pickaxe: Much faster on stone (uses durability)

Crafting

Press I to open inventory and crafting grid. Place items in the grid and click the result to craft!

Placing Blocks

Select an item from hotbar (1-9 keys) and right-click to place it in the world.

Smelting

Craft a Forge (5 stone blocks), right-click it, add fuel and items to smelt!

Saving

Your progress auto-saves. You can also manually save from the menu. 3 save slots available.

📚 Crafting Recipes

Basic Resources

  • Planks: 1 Wood → 4 Planks (shapeless)
  • Sticks: 2 Planks vertical → 4 Sticks

Tools

  • Axe: 2 Planks + 2 Sticks (shaped, symmetrical)
  • Pickaxe: 3 Planks horizontal + 2 Sticks vertical
  • Knife: 2 Planks + 1 Stick vertical

Structures

  • Crafting Table: 4 Planks 2x2
  • Forge: 5 Stone (cross pattern)

Tool Durability

  • Axe: 60 uses
  • Pickaxe: 50 uses
  • Knife: 40 uses
