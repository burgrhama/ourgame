## DEPLOYMENT OPTIONS:

### Option 1: Local Play
Run on your computer:
```bash
python main.py                    # Full desktop game
# OR
Open index.html in browser        # Web version
```

### Option 2: GitHub Pages (FREE!)
1. Create GitHub account (github.com)
2. Create new repo: `pixel-survival-game`
3. Push this folder to GitHub:
   ```bash
   git init
   git add .
   git commit -m "Pixel survival game"
   git remote add origin https://github.com/YOUR_USERNAME/pixel-survival-game.git
   git push -u origin main
   ```
4. Go to Settings → Pages → Enable
5. Game goes live at:
   ```
   https://YOUR_USERNAME.github.io/pixel-survival-game/
   ```
   Your website automatically serves `index.html`!
